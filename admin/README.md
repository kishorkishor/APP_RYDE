
# Riyadh Admin Portal

React/Vite admin portal for the Riyadh ride operations dashboard.

## Local development

Run `npm i` to install dependencies.

Run `npm run dev` to start the frontend.

Set `VITE_ADMIN_API_BASE_URL` to the backend API URL. Locally this is usually `http://localhost:3000`.

## Netlify

- Build command: `npm run build`
- Publish directory: `dist`
- Environment variable: `VITE_ADMIN_API_BASE_URL=https://your-live-backend-url`
